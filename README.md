# AI Real Estate Valuator

## Projet en cours

### Objectif
Ce projet a pour but de créer une application capable de prédire le prix de l'immobilier en utilisant le Machine Learning.

### Roadmap (Plan d'action)
- [ ] Récupération des données (Scraping ou Dataset Kaggle)
- [ ] Nettoyage et exploration des données (Pandas/Seaborn)
- [ ] Création du modèle prédictif (Scikit-Learn)
- [ ] Création de l'API (Flask)
- [ ] Visualisation des résultats
